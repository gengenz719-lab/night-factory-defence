# アイデア置き場(レリック案・敵案)

実装済みかどうかに関わらず、思いついたらここに追記する。
実装するときは config.js にデータとして追加する。

## レリック案

### プレイヤー強化

```text
跳弾: 弾が敵を貫通する(実装済み)
速射: 攻撃速度+30%(実装済み)
俊足: 移動速度+20%(実装済み)
硬い体: 最大HP+30、全回復(実装済み)
背水: HPが減るほど攻撃速度上昇
炎上弾: 近接攻撃に炎上付与
```

### 拠点強化

```text
砲術: タレット射程+25%(実装済み)
連射砲: タレット発射速度+30%(実装済み)
強化壁: 壁の最大HP+50%(実装済み)
自己修復壁: 壁が昼の間に自動回復する
節約術: 弾薬消費-30%、タレット射程-10%(デメリット付き)
```

### 工場強化

```text
効率炉: 加工炉の弾薬生産+1(実装済み)
血の採掘: 採掘速度+50%(実装済み)
弾薬箱: 即座に弾薬+80(実装済み)
ゾンビ燃料: 倒した敵が資源になる
暴走炉: 生産速度+80%、たまに火災発生(デメリット付き)
夜勤: 夜の間だけ生産速度+50%
```

## 敵案

```text
ウォーカー: 遅い雑魚。数で圧をかける(実装済み・コスト1)
ランナー: 速い。プレイヤーの反応を要求(実装済み・コスト2)
タンク: 硬い。DPSチェック(実装済み・コスト5)
自爆ゾンビ: 壁に到達すると爆発。壁の配置を考えさせる
スピッター: 遠距離攻撃。壁の内側から撃ってくる → 位置取りを要求
サモナー: 雑魚を召喚するエリート。優先処理を要求(コスト8)
掘りゾンビ: 壁を無視して地面から出てくる(終盤用)
```

## 弾薬バリエーション案(工場が武器ビルドになる)

```text
鉄 → 通常弾
鉄 + 火薬 → 爆発弾
鉄 + 毒液 → 毒弾
鉄 + 電池 → 電撃弾
```
