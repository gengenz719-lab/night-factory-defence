# ゲームの憲法

このプロジェクトで絶対にブレさせない方針。仕様の迷いが出たら、ここに立ち返る。

## 一文コンセプト

```text
2〜4人で拠点を作り、昼に小さな生産ラインを組み、夜にゾンビウェーブを防衛する軽量ローグライト工場ゲーム。
```

## 3つの言い換え

- Factorioを作るのではなく、**防衛を楽にする小さな工場**を作る
- Vampire Survivorsを作るのではなく、**Waveごとに拠点ビルドが変わるレリック**を作る
- ゾンビサバイバルを作るのではなく、**友達と夜を乗り切る拠点防衛**を作る

## 守ること

1. 1ランは20〜30分で終わる。ライトに遊べること最優先
2. 工場は自陣エリア内で完結する小規模なもの
3. マルチ(1〜4人)を将来入れる前提でコードを設計する
4. 完成より「毎週遊べる状態」を維持する
5. 初心者メンバーが読めないコードは書かない

## 勝敗条件(固定)

- 勝利: 10 Wave 生存
- 敗北: 拠点コア破壊(プレイヤー死亡は敗北ではない → リスポーン)

## 難易度設計の原則

- 敵のHP・攻撃力は控えめに上げ、**スポーン数と組み合わせ**で難易度を作る
- 敵はコスト制。Waveごとに敵予算を決めて編成する
- 強いWaveのあとに休憩Waveを入れる(右肩上がりにしない)
- 新しい敵は少量から紹介する
- ボスWaveは数値ではなくギミックで山場を作る
