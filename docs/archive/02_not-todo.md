# やらないことリスト

「面白そうだから」で追加しがちだが、今はやらないと決めたもの。
追加したくなったら、まずここで議論する(Discordの #アイデア置き場 へ)。

## プロトタイプ段階(現在)でやらないこと

```text
オンラインマルチ
大量の敵種類(3種+ボスで十分)
高度な工場(流体・電力網・物流ロボット・列車)
ストーリー
複雑なクラフト(多段階レシピ)
Steam実績・ランキング
セーブ機能
サウンド・BGM(遊びが固まってから)
凝ったグラフィック(図形描画で十分)
```

## 製品版でも(最初は)やらないこと

```text
巨大マップ探索
5人以上のマルチ
モバイル対応
ローカライズ(日英2言語まで)
Mod対応
```

## 判断基準

- 「防衛を楽にする小さな工場」から外れる工場要素は入れない
- 1ラン20〜30分を超えさせる要素は入れない
- 実装より先に「それがないと何がつまらないか」を言語化する
