# プロトタイプ仕様(Prototype 01: Night Factory Defence)

目的: **ゾンビウェーブを、プレイヤー戦闘 + 簡易工場 + タレット供給で防ぐ面白さを確認する。**

技術: HTML5 + Canvas 2D + 素のJavaScript(ビルド不要、index.htmlを開くだけ)

## 設計メモ(なぜこの数値か)

- **Wave予算**: `[10,15,22,30,42,38,52,68,85,120]` — 開発メモの難易度カーブ。Wave6は意図的な休憩、Wave10はタンク4体確定+予算120の山場
- **弾薬経済**: 採掘機1台(鉄0.5/秒) + 加工炉1台(弾薬1/秒) ≒ タレット1基の連射(2/秒)の半分を賄う。**タレットを増やすほどラインの増設が必要になる**のがこのゲームの核
- **敵はステータスより数と組み合わせ**: 敵HPは控えめ、スポーン間隔0.9秒で圧を作る
- 検証済みバランス: 自動プレイ(タレット増設のみ・プレイヤー射撃なし)でWave9まで生存、Wave10で敗北。**プレイヤーが戦闘参加すればWave10を越えられる**難易度帯に着地している

## 数値を変えたいとき

すべて `prototype/js/config.js` にある。ロジックを触らずに調整できる:

```text
WAVE_BUDGETS   … 難易度カーブそのもの
ENEMIES        … 敵のHP/速度/コスト/登場Wave
BUILDINGS      … 建物のコスト/HP/生産速度/射程
RELICS         … レリックの追加はここに1エントリ足すだけ
```

## 動作確認の手順(変更したら最低これだけ)

```text
1. index.html をブラウザで開く(またはリロード)
2. F12のConsoleに赤いエラーが出ていないこと
3. ゲーム開始 → タレット設置 → Spaceで夜 → 敵が湧いてタレットが撃つこと
4. Wave1クリアでレリック3択が出ること
```
