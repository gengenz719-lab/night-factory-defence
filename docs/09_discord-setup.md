# Discord 開発サーバー 設計図

このプロジェクト専用のDiscordサーバーの作り方と、そのままコピペで使えるチャンネル構成。
少人数のパートナー開発 + AIバイブコーディングに合わせた最小構成。

## 0. サーバーを作る

1. Discord左の「+」→「オリジナルの作成」→「自分と友達のため」
2. サーバー名: `Kagyo games / Night Factory Defence`(好みで)
3. 作成したら、下の「1. カテゴリとチャンネル」を作っていく

> チャンネル作成: カテゴリ名の横の「+」から。各チャンネルの「説明(トピック)」は
> チャンネル設定(歯車)→「概要」→「チャンネルトピック」に貼る。

## 1. カテゴリとチャンネル

### 📋 はじめに
| チャンネル | 種類 | トピック(説明欄に貼る) |
|---|---|---|
| `welcome` | テキスト | このサーバーの入口。まず固定メッセージを読む |
| `ルール` | テキスト | 共同開発の約束ごと。作業前に一読 |
| `リンク集` | テキスト | リポジトリ・遊べるリンク・ドキュメントの置き場 |

### 🎮 開発
| チャンネル | 種類 | トピック |
|---|---|---|
| `今日やること` | テキスト | 作業を始める前に「何を触るか」を宣言(ファイル衝突防止) |
| `できた報告` | テキスト | スクショ・動画で成果を共有。いちばん大事なチャンネル |
| `バグ報告` | テキスト | バグは敵キャラ扱い。下のフォーマットで報告 |
| `アイデア置き場` | テキスト | レリック案・敵案・やりたいこと |
| `aiに聞いたこと` | テキスト | うまくいったAIプロンプトの共有 |

### 💬 雑談
| チャンネル | 種類 | トピック |
|---|---|---|
| `雑談` | テキスト | なんでも |
| `参考ゲーム` | テキスト | 参考になるゲーム・動画のリンク |

### 🔊 通話
| チャンネル | 種類 | 用途 |
|---|---|---|
| `もくもく作業` | ボイス | 画面共有しながら一緒に開発 |
| `テストプレイ` | ボイス | みんなで最新版を遊ぶ |

## 2. 固定メッセージ(コピペ用)

### `welcome` に貼って固定(ピン留め)する

```text
🏭 Night Factory Defence 開発サーバーへようこそ

2〜4人で拠点を作り、昼に工場ラインを組み、夜にゾンビウェーブを防ぐ協力ローグライト工場ゲームを作っています。

▶ いますぐ遊ぶ(インストール不要):
https://gengenz719-lab.github.io/night-factory-defence/

▶ ソースコード:
https://github.com/gengenz719-lab/night-factory-defence

はじめての人へ:
1. まず上のリンクで遊ぶ
2. #ルール を読む
3. #今日やること で「今日やること」を宣言
4. できたら #できた報告 にスクショを貼る

プログラミング経験がなくてもOK。AIが3人目の開発メンバーです。
```

### `ルール` に貼って固定する

```text
📏 共同開発 7か条

1. 同じファイルを同時に触らない(#今日やること で宣言してから作業)
2. 数値調整だけなら prototype/js/config.js だけ触ればいい(壊れにくい)
3. 作業前に必ず最新版を取得する(git pull)
4. 作業後は「何を変えたか」を #できた報告 に書く
5. AIが書いたコードは必ず人間が一度読む
6. 動かない状態で共有(push)しない
7. 毎週1回、最新版をみんなで遊ぶ

困ったらAIに聞く or #雑談 で相談。GitHubに履歴が全部残るので、壊しても必ず戻せます。
```

### `リンク集` に貼って固定する

```text
🔗 リンク集

遊ぶ:      https://gengenz719-lab.github.io/night-factory-defence/
リポジトリ: https://github.com/gengenz719-lab/night-factory-defence
参加ガイド: リポジトリの docs/05_team-onboarding.md
GitHub手順: リポジトリの docs/08_github-guide.md
```

### `バグ報告` に貼って固定する(報告テンプレ)

```text
🐛 バグ報告テンプレ(コピーして使ってね)

バグ名:
症状:
危険度: ★〜★★★★★
発生条件:
討伐担当:(やりたい人が名乗る)
```
