# 開発ロードマップ

## 全体の段階

```text
第1段階: 戦闘だけ        ✅ プロトタイプで実装済み
第2段階: 建築だけ        ✅ プロトタイプで実装済み(壁・タレット・設置UI・資源消費)
第3段階: 軽工場          ✅ 採掘機・加工炉・弾薬供給まで実装済み(ベルトは未)
第4段階: ローグライト     ✅ Wave後3択レリック・10Wave構成・リザルト実装済み
第5段階: マルチ          ⬜ ローカル協力 → Steamオンライン
第6段階: Steam用         ⬜ エンジン移植 → ストアページ → デモ → Next Fest
```

プロトタイプが最初から第4段階まで到達しているため、当面は
**「遊んで、面白さを削り出す」フェーズ**。数値調整・レリック追加・敵追加を毎週回す。

## 週次クエスト方式

毎週テーマを決めて、週末に30分遊ぶ。

```text
30分作業 → 30分テスト → 30分笑ったバグ整理 → 30分次のクエスト決め
```

### 直近のクエスト候補(難易度つき)

| クエスト | 難易度 | 内容 |
|---|---|---|
| バランスの神になる | ★☆☆☆☆ | config.js の数値だけ触って10Waveの体感を調整 |
| レリックを5個増やす | ★★☆☆☆ | config.js にレリック定義を追加(効果は既存modsの組合せ) |
| 修理機能 | ★★☆☆☆ | 昼に壁を右クリック長押しで鉄を消費して修理 |
| 新しい敵「自爆ゾンビ」 | ★★★☆☆ | 壁に到達すると爆発して範囲ダメージ |
| ベルトコンベア | ★★★★☆ | 採掘機→加工炉→タレットへ物理的にアイテムを流す |
| ボスギミック | ★★★★☆ | Wave10のボスが雑魚を召喚する |
| ローカル2人協力 | ★★★★★ | 2つ目のプレイヤーをゲームパッドで操作 |

## エンジン移植のタイミング

以下が揃ったらGodot(またはUnity)への移植を開始する。詳細は 04_engine-decision.md。

```text
✅の条件:
- 2人以上で遊んで「もう1回」と言われる状態
- レリック15個以上でビルドの多様性が確認できた
- ベルトを入れるか入れないかの結論が出た
```

## Steamに向けて

```text
完成してから売るのではなく、
ストアページ公開 → デモ公開 → ウィッシュリスト獲得 → Steam Next Fest
の順で早めに露出を始める。

デモの合格ライン:
- 10〜20分で1ラン終わる
- 2人で遊ぶと面白い
- スクショで何のゲームかわかる
```
