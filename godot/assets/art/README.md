# Godot版オリジナルアート

このフォルダはGodot版専用に新規制作したゲームアセットを格納する。Unity版の素材は流用しない。

## vehicle/survival_vehicle.png

- 用途: 横視点の断面車両
- 制作日: 2026-07-10
- 制作方法: OpenAI組み込み画像生成 + クロマキー透過処理
- アート基準: `docs/art/godot-art-direction-v1.png`
- ゲーム内配置: 中心 `(800, 510)`、表示倍率 `0.62`
- 車内移動、修理地点、敵の攻撃判定は既存のGodot座標系を維持する

生成元のクロマキー画像は実行時に不要なため、最終透過PNGのみ管理する。

## actors/

- `crew_survivor.png`: プレイヤー本体
- `crew_rifle.png`: マウス照準に追従する分離武器
- `enemy_walker.png`: 標準ゾンビ
- `enemy_runner.png`: 高速ゾンビ
- `enemy_climber.png`: 屋根を狙う長腕変異体

全素材は同一の生成シートから切り出し、連結成分抽出で隣接素材の混入を除去している。キャラクターと武器を分離することで、静止画アセットでも照準方向をゲーム側で表現する。
